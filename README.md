# GitHub User Explorer

A web application for searching GitHub users built with React + TypeScript + Vite.

## Features

- Search GitHub users by username
- Display user info: avatar, name, bio, repos, followers, following
- Display last 5 repositories with star count and links
- Error handling: user not found (404) and network errors
- Share button that copies URL with username as query parameter
- Responsive layout

## Tech Stack

- React + TypeScript
- Vite
- Axios
- TanStack Query (React Query)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
